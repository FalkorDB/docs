# Similarity Functions

FLEX similarity utilities provide functions for measuring similarity between data structures. These functions are useful for comparing sets, finding duplicates, and implementing recommendation systems.

## Available Functions

| Function | Description |
|----------|-------------|
| [similarity.jaccard](./jaccard.md) | Calculate Jaccard similarity coefficient between two sets |

## Common Use Cases

- **Recommendation Systems**: Find similar items based on shared attributes
- **Duplicate Detection**: Identify similar or duplicate records
- **Set Comparison**: Measure how similar two sets of items are
- **Content Similarity**: Compare collections of tags, categories, or features
