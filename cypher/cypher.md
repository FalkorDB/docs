# Cypher clauses

Cypher query consists of 1 or more clauses

* [MATCH](/match)
* [OPTIONAL MATCH](/optional-match)
* [WHERE](/where)
* [RETURN](/return)
* [ORDER BY](/order-by)
* [SKIP](/skip)
* [LIMIT](/limit)
* [CREATE](/create)
* [MERGE](/merge)
* [DELETE](/delete)
* [SET](/set)
* [WITH](/with)
* [UNION](/union)
* [UNWIND](/unwind)
* [FOREACH](/foreach)
* [CALL {}](/call-)
